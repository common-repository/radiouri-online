Plugin cu widget pentru afisarea posturilor de radio online preferate.

== Description ==
Acest plugin cu widget va permite sa alegeti posturile de radio online pe care doriti sa le afisati intr-un widget. Sunt prezente 10 categorii muzicale (in curand mai multe) din care puteti alege maxim 5 posturi per categorie. Genurile muzicale din care puteti alege sunt: Anii 80-90, Crestina, Clasica, Dance, Dragoste, Hip Hop, House, Latino, Manele, Orientala, Populara, Relaxare, Rock.

== Installation ==
Pasii de urmat pentru instalarea pluginului:

    1. Descarcati pluginul, dezarhivati si urcati directorul radiouri-online in folderul /wp-content/plugins/.
    2. Intrati in partea de administrare a blogului, Plugins si activati pluginul Radio Online
    3. In Settings veti gasi butonul Radio Widget de unde puteti administra categoriile si radiourile care doriti sa fie afisate pe blogul vostru
    4. Daca doriti sa afisati posturile din anumite categorii bifati categoriile respective si apoi apasati butonul Listeaza Posturi. Din lista afisata bifati posturile care doriti sa le afisati in sidebarul blogului si apoi bifati "Salveaza Posturi".
    5. Acum navigati la Appearance -> Widgets si alegeti pozitia widgetului in sidebar.
